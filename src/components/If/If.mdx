# If

Conditional rendering utility component to show or hide content based on `condition` boolean.

## Properties

<PropsTable of={If} />

## Render Parameters
Name | Type | Description | Default
--- | --- | --- | ---
children | [React Child Elements](https://reactjs.org/docs/react-api.html#reactchildren) | Component children to be shown / hidden based on condition | `null`

## Examples

```jsx
import { If } from 'hoc-toolbox';
```

<Playground>
  <If condition={true}>
    <div>You can see me!</div>
  </If>

  <If condition={false}>
    <div>You can't see me</div>
  </If>
</Playground>
